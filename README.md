# schissel-practice-ops

A **private** telemedicine practice-operations dashboard for Schissel Medical Group.

## Purpose

This project provides internal tooling and operational visibility for a telemedicine practice, including:

- Scheduling and appointment management
- Provider availability tracking
- Billing and claims status monitoring
- Operational metrics and reporting

## Security & Compliance

This repository is **private**. All data handling must comply with HIPAA requirements. PHI must never be committed to this repository. See [CLAUDE.md](CLAUDE.md) for AI-assistant constraints regarding PHI boundaries.

## Getting Started

```bash
npm install
npm run dev
```
